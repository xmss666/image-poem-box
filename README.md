# 意象·诗匣

一款专注于中国古典诗词意境探索的交互式网页应用，让用户通过自由组合古典意象，激发灵感，生成独特的诗意描写。

## 🎯 项目特色

- **丰富的意象数据库**：6大类别，216个精选古典意象
- **智能诗意生成**：基于AI的古典诗词意境创作
- **意境配图生成**：自动生成符合诗词风格的水墨插图
- **诗签导出功能**：将作品合成为精美图片
- **响应式设计**：完美适配桌面、平板和手机
- **本地数据持久化**：自动保存用户的创作记录

## 🎨 意象分类

### 自然景物
明月、残阳、孤云、寒鸦、暮雪、春草、秋叶、空山、松风、竹影等天地自然中的客观物象

### 建筑空间  
小楼、西窗、危栏、柴门、玉阶、空城、禅寺、驿站等人造或半人造的物理场所

### 器物用具
红笺、酒盏、金钗、香炉、残烛、琵琶、玉笛等与日常生活相关的具体物件

### 时间节令
更漏、钟声、子规、砧声、寒食、清明、七夕等表示时间流逝的符号

### 情感状态
离歌、泪痕、旧梦、思亲、怀古、断肠等直接表达内心情绪的词汇

### 生命行为
独坐、凭栏、望归、忆远、听雨、吹笛等人在特定情境中的姿态动作

## 🚀 功能介绍

### 1. 意象选择
- 点击分类标签切换不同类别的意象
- 点击意象按钮添加到创作区（最多7个）
- 支持自定义意象输入（不超过4字）
- 可以单独删除已选意象或清空全部

### 2. 诗意生成
- 选择2-7个意象后点击"生成意境"
- AI将根据固定模板生成100-150字的诗意文本
- 自动配以经典诗词名句作为装饰

### 3. 配图生成
- 生成诗意后点击"生成配图"
- 根据诗意内容自动创作古典水墨风格插图
- 支持多次生成以获得不同视觉表现

### 4. 诗签导出
- 配图生成后点击"导出诗签"
- 自动合成意境文本、配图和意象标签
- 生成高清PNG格式图片，适合分享与收藏

## 🎨 设计风格

- **色彩方案**：米白、赭石、墨黑三色主调
- **字体选择**：宋体、楷体等具有中式传统韵味的字体
- **视觉风格**：新中式古典美学，融合传统水墨画意境与现代极简主义

## 📱 使用说明

1. **选择意象**：在顶部分类标签中选择类别，然后点击意象按钮添加到创作区
2. **自定义添加**：在自定义输入框中输入个性化意象（最多4字）
3. **生成诗意**：选择至少2个意象后点击"生成意境"按钮
4. **生成配图**：诗意生成成功后点击"生成配图"按钮
5. **导出作品**：配图生成完成后点击"导出诗签"下载图片

- **前端技术**：HTML5 + CSS3 + JavaScript (ES6+)

## 🔧 部署说明

### 快速部署

本项目为纯静态网页应用，提供多种部署方式：

#### 1. GitHub Pages 部署（推荐）
```bash
# 1. 创建 GitHub 仓库并上传代码
# 2. 在仓库设置中启用 GitHub Pages
# 3. 选择分支并保存
# 4. 访问 https://your-username.github.io/image-poem-box
```

#### 2. 本地服务器启动
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx live-server --port=8000

# 使用 npm
npm install
npm start
```

#### 3. 静态文件服务器
- 将所有文件上传到 Web 服务器
- 确保服务器支持 HTTPS（推荐）
- 访问 index.html 即可使用

### 部署文件说明

- `index.html` - 主页面文件
- `styles.css` - 样式文件
- `script.js` - 功能脚本
- `package.json` - 项目配置
- `deploy-guide.md` - 详细部署指南
- `CNAME` - 自定义域名配置（可选）

### 环境要求
- 现代浏览器（Chrome 60+, Firefox 55+, Safari 12+, Edge 79+）
- 支持 HTTPS（API 调用需要）
- 网络连接（AI 服务调用）

## 📝 注意事项

- 当前版本使用模拟API演示功能
- 实际部署时需要配置真实的AI生成服务
- 图片生成功能需要接入相应的图像生成API
- 建议在生产环境中添加错误处理和用户反馈机制
